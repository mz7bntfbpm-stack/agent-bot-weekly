Footer-Update für agent-bot.de

Anleitung zur Integration der rechtlichen Links

Diese Seite zeigt, wie der Footer der Hauptseite aktualisiert werden muss, um die erforderlichen Links zu den rechtlichen Seiten hinzuzufügen. Der Footer sollte die Links zu Impressum, Datenschutz, AGB und Widerruf enthalten.

Aktueller Footer (ohne Links)

<footer class="footer">
  <p class="copyright">
    © 2024 AGENT-BOT / CRM.BUILD
  </p>
</footer>

Neuer Footer (mit allen Links)

<footer class="footer">
  <div class="footer-content">
    <div class="footer-links">
      <a href="impressum.html">
        Impressum
      </a>
      <a href="datenschutz.html">
        Datenschutz
      </a>
      <a href="agb.html">
        AGB
      </a>
      <a href="
        widerrufsbelehrung.html">
        Widerruf
      </a>
    </div>
    <p class="copyright">
      © 2024 AGENT-BOT / CRM.BUILD
    </p>
  </div>
</footer>

Schritt 1: CSS hinzufügen

Fügen Sie diese CSS-Stile zu Ihrer bestehenden CSS-Datei oder in einen <style>-Block im <head>-Bereich hinzu:

.footer {
  background: #1a1a2e;
  color: #ffffff;
  padding: 40px 24px;
}

.footer-content {
  max-width: 1200px;
  margin: 0 auto;
  text-align: center;
}

.footer-links {
  display: flex;
  justify-content: center;
  flex-wrap: wrap;
  gap: 24px;
  margin-bottom: 24px;
}

.footer-links a {
  color: #ffffff;
  text-decoration: none;
  font-size: 14px;
  opacity: 0.8;
  transition: all 0.2s ease;
}

.footer-links a:hover {
  opacity: 1;
  text-decoration: underline;
}

.copyright {
  font-size: 14px;
  opacity: 0.6;
}

Schritt 2: Footer-HTML ersetzen

Ersetzen Sie den bestehenden Footer-Code durch den neuen Code mit allen erforderlichen Links.

Schritt 3: Rechtliche Seiten hochladen

Laden Sie alle erstellten HTML-Dateien auf Ihren Server hoch:

  • impressum.html
  • datenschutz.html
  • agb.html
  • widerrufsbelehrung.html
  • styles.css (falls noch nicht vorhanden)

Schritt 4: Cookie-Banner integrieren

Fügen Sie den Cookie-Consent-Banner in Ihre Hauptseite ein. Fügen Sie diesen Code vor dem schließenden </body>-Tag hinzu:

<script src="cookie-consent.js"></script>

Wichtig: Firmendaten ergänzen

Vergessen Sie nicht, in allen erstellten Dateien die Platzhalter durch Ihre tatsächlichen Firmendaten zu ersetzen:

  • [Firmenname eintragen] → Ihr vollständiger Firmenname
  • [Rechtsform eintragen] → z.B. GmbH, UG, Einzelunternehmen
  • [Straße und Hausnummer] → Ihre vollständige Adresse
  • [Postleitzahl und Ort] → PLZ und Stadt
  • [E-Mail-Adresse] → Ihre Kontakt-E-Mail
  • [Telefonnummer] → Ihre Telefonnummer
  • [HRB-Nummer oder vergleichbar] → Handelsregisternummer
  • [DE-Nummer eintragen] → Umsatzsteuer-ID
